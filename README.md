# UniversalCam-AntennaFinder
This is base code application I've created to assist with finding multiple different manufacturer devices. This can be used to find ubiquities, Axis, Bosch etc.
Work is still being done, currently only a basic CLI applciation. Plan to integrate filters, specific selection and an easy to navigate GUI

Please go to https://github.com/areja02/UniversalCam-AntennaFinder/releases/tag/Executables for the executable download as well as source code for download.
